
# 🎮 Number Guessing Game

A clean, terminal-based number guessing game implemented in C++. Players select a difficulty tier, and the program dynamically adjusts the maximum allowed attempts while tracking player performance.

[![Roadmap.sh Project](https://img.shields.io/badge/Roadmap.sh-Project-blue)](https://roadmap.sh/projects/number-guessing-game)
[![Language](https://img.shields.io/badge/Language-C%2B%2B-orange)](https://en.cppreference.com/)

---

## 🚀 Features

* **Dynamic Difficulty Engine:** Choose between three gameplay modes tailored to your skill level:
  * **Easy:** 10 attempts
  * **Medium:** 5 attempts
  * **Hard:** 3 attempts
* **Intelligent Feedback:** Higher/lower contextual hints provided after each incorrect guess.
* **Score Tracking:** Tracks the exact number of attempts taken to reach the correct answer.
* **Seamless Replayability:** Built-in game loop allowing players to restart instantly without restarting the executable.

---

## 🛠️ Tech Stack & Requirements

* **Language:** C++ (Standard ISO/IEC 14882)
* **IDE/Compiler:** Visual Studio (MSVC) or any standard C++ compiler (GCC, Clang)

## 📦 Installation & Setup

To run this project locally using **Visual Studio**:

1. **Clone the Repository:**
```bash
git clone [https://github.com/hosoazizi/number-guessing-game.git](https://github.com/hosoazizi/number-guessing-game.git)
```
Open Visual Studio.
Select Open a local folder and choose the directory where you cloned the project.
Open main.cpp, press Ctrl + F5 to compile and run the game.
